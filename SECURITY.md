# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any BlackRoad OS product, please report it responsibly.

**Email:** security@blackroad.io

**Do NOT** open a public GitHub issue for security vulnerabilities.

## Response Timeline

- **Acknowledgment:** Within 24 hours
- **Initial Assessment:** Within 72 hours
- **Fix & Disclosure:** Coordinated with reporter

## Scope

This policy applies to all repositories under [BlackRoad-OS-Inc](https://github.com/BlackRoad-OS-Inc).

## Supported Versions

Only the latest version on the `main` branch is supported with security updates.
